## Projetos em destaque

### FinanceOS — Plataforma SaaS White Label

Plataforma multi-tenant para monitoramento financeiro e investimentos, construída com arquitetura escalável e reutilizável.

**Front-End**
- React + TypeScript + Vite
- Gerenciamento de estado com React Query e Zustand
- Estrutura modular baseada em domínio
- Sistema white-label com resolução por DNS

**Back-End**
- APIs REST com NestJS
- Autenticação stateless com JWT
- Multi-tenant via header (`X-Tenant-ID`)
- Isolamento de dados por `tenant_id`
- Integração com PostgreSQL via Prisma
